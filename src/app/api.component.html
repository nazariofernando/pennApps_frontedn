Our API was built so that anyone can use the power of genetic algorithms and (in the near future) machine learning
The first step is to subscribe on our site:
Sign Up
If you are logged in, your email should appear here:
{{currentUser.email}}
Now that you are all set, let's begin.
First, include in your code a JSON object with the following structure:
{{ jsonExample | json }}

email is the email you used to log in
gen is the number (an integer) of generations you want to create
ind is the number (an integer) of individuals you want to have per generation
params are the variables you have in your model (and their initial values) that are going to mutate
percentages is by how much you want to mutate your variables
min and max are the minimun and maximun values for each param (if there are none, just leave them 0)
evaluation is the equation which evaluates how good is an individual, according to your model. It must be a string (that is, inside the quotation marks) and it must the model's variables as params['variableX']. It accepts any function and value from javascript Math library
State is the direction you want us to follow. If it is the boolean value true, the algorithm will work so that increases the evaluation. If it is false, it will decrease the evaluation
After you create your json object you call a http post request on the following address: https://darwindesignsapi.herokuapp.com/api, and send your json data
